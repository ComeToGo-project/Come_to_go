# 📌 shared 폴더 사용 규칙 (main 브랜치)

이 폴더는 **최종 확정된 공용 문서**만 보관합니다.  
개발 중이거나 수정될 가능성이 있는 자료는 `develop/shared`를 사용하세요.  

---

## ✅ 업로드 대상
- 기획서 확정본 (PDF, PPT 등)  
- 스토리보드 최종본  
- 잡리스트 최종본  
- 그 외 변경될 일 없는 공식 문서  

---

## ❌ 업로드 금지
- 초안, 임시 파일  
- 수정 가능성이 있는 자료  
- 개인 작업 파일 (→ `members/` 폴더 사용)  

---

## 📝 커밋 메시지 예시
- `기획서 최종본 main/shared 업로드`  
- `스토리보드 확정본 추가`  
- `잡리스트 최종 문서 업로드`
